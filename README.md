 ## Goals

The main goal of this challenge is to practice HTML and vanilla CSS, making 30 mini projects a day to help you improve your current skills and learn new ones.

## Rules

* Make one project a day.
* Share your progress through social media (Twitter, Linkedin, etc) using #30daysOfCSS;
* Project must be finished until 11:59PM.
